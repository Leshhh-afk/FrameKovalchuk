Данная программа написана на Java, предназначена для хранения и обработки информации о котах и их владельцах.
Программа работает с базой данных PostgreSQL, в которой хранятся следующие данные:
информация о котах (кличка, возраст, порода);
информация о владельцах котов (логин, пароль);
связь между котом и его владельцем.
В рамках лабораторной работы программа и база данных запускаются в отдельных Docker-контейнерах с использованием Docker Compose
